1.Project Description

This project is about Iris Flower Classification using Machine Learning.
The goal is to predict the type of iris flower based on its features like sepal length, sepal width, petal length, and petal width.
The dataset used is the Iris dataset, which is a popular dataset for beginners in Data Science and machine learning.

2.Technologies Used
Python
Scikit-learn
Machine Learning (Logistic Regression)

3.Dataset
The dataset contains 150 records of iris flowers with 3 classes:
Iris-setosa
Iris-versicolor
Iris-virginica

*Each record has 4 features:
Sepal Length
Sepal Width
Petal Length
Petal Width
